RetroCons

About

RetroCons is an icon font made for typing input prompts for all consoles from the past 40+ years as a light-weight font. The font is also open-source on GitHub, so if you'd like to check that out, please do so here:

Using the Font

First, download the latest version of the font.

Unzip the file and drag it in to the root of your project's file directory. and name it retrocons. Next, add the following CSS links to the head section of your HTML file. 

/* CSS Links */
<link rel="stylesheet" href="/retrocons/css/animation.css">
<link rel="stylesheet" href="/retrocons/css/retrocons-codes.css">
<link rel="stylesheet" href="/retrocons/css/retrocons-embedded.css">
<link rel="stylesheet" href="/retrocons/css/retrocons-ie7-codes.css">
<link rel="stylesheet" href="/retrocons/css/retrocons-ie7.css">
<link rel="stylesheet" href="/retrocons/css/retrocons.css">

Using the Icons

In order the use the icons, you need to give an element a specific class. That class is the icon id. You may do this to a span element, or i, whichever you prefer.
Here's an example, using the icon that represents . 

<!-- Example Code -->
<p>
    <i class="icon-btn-a"></i> - Jump
</p>

Result:

- Jump

General Usage

In order to just use the font normally outside of the browser, you can just copy the Unicode for that icon. The Unicode for all the icons can be found here.
